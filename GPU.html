Overview: The Graphics Processing Unit

The Graphics Processing Unit displays imagery on the monitor screen in the simplest terms. A user may find themselves requiring a stronger GPU depending on their usage; if the user plays extensive games or does a lot of rendering, they might find themselves requiring a higher-end model. Inversely, if the user exclusively uses a computer for web browsing or other documenting work, they might find themselves perfectly content with a lesser-end GPU or none at all, with just the CPU alone being able to carry their workload.
