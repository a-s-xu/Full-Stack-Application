Overview: The Central Processing Unit

The Central Processing Unit is equivalent to the human nervous system. The Central Processing Unit is able to access every component that is connected to it via the motherboard. Only through the CPU can the computational functions of the rest of the computer be utilized together in one cohesive mass; otherwise, it's just tech parts stuck together.
